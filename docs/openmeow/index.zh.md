# OpenMeow

OpenMeow 是一个纯 Swift macOS 语音 API 网关，使用本地引擎（sherpa-onnx、WhisperKit、Qwen3-TTS）提供 OpenAI 兼容的 TTS/ASR 接口。

## 特性

- 兼容 OpenAI 的 TTS 和 ASR REST API
- 多引擎支持：sherpa-onnx、WhisperKit、speech-swift
- 模型注册中心，支持自动下载
- 云服务商集成（阿里云、MiniMax、腾讯云）
- 菜单栏内存使用监控 UI
